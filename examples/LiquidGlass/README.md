# Liquid Glass Example

This original example shows a superellipse panel, an overlapping rounded
panel, a Liquid Glass button, and a spring-smoothed slider.

The release artifact `liquid-glass-example-X.Y.Z.zip` is self-contained: it
includes the example project and the matching addon under
`addons/liquid_glass/`. Extract it, open `project.godot` in Godot 4.7 Mono,
build, and run. After the first successful build, enable **GodotUI Liquid
Glass** in Project Settings > Plugins to add its custom types to the Create
Node dialog.

Inside the source repository the example project links the addon's C# runtime
files for compilation. Use the packaged example for the exact clean release
layout.
